<br>
손쉽게 휴게소 정보를 얻고 추천받을 수 있는 HUEAT에 오신것을 환영합니다.

## 팀소개 및 담당기능
- ### 송승경  :cherry_blossom:
  - 담당 페이지 : 메인화면, 공지사항, 소개글, 이벤트, 고객센터, 쇼핑몰
  - 상세 기능 : 공지사항&이벤트,고객후기,고객문의 추가/수정/삭제, 쇼핑몰 추가, 삭제기능, 소개글 작성, 메인화면 디자인작업
- ### 조석훈  :pineapple:
  - 담당 페이지 : 휴게소 찾기, 휴게소 목록
  - 상세 기능 : 지도API, 휴게소 검색, 휴게소 추가/수정/삭제
- ### 홍재희 :tangerine:
  - 담당 페이지 : 휴게소 상세페이지
  - 상세 기능 : sns공유하기API, 휴게소별 푸드코트 메뉴 평점, 휴게소별 평점, 지도API
- ### 차유지  :melon:
  - 담당 페이지 : 주문결제, 회원가입, 로그인, 휴게소 즐겨찾기
  - 상세 기능 : 결제API, 회원가입 및 로그인/로그아웃, 휴게소 즐겨찾기 추가 및 삭제 
- ### 손범수 :koala:
  - 담당 페이지 : 마이페이지, 관리자페이지
  - 상세 기능 : 비밀번호확인, 회원정보수정, 나의활동(Q&A,REVIEW), 회원탈퇴, 관리자페이지 나의활동(Q&A,NOTICE,EVENT)
