## 팀소개 및 담당기능
- 송승경 : 메인화면, 공지사항, 소개글, 이벤트, 고객센터, 쇼핑몰
  - 담당 기능 : 
- 조석훈 : 휴게소 찾기, 휴게소 목록
  - 담당 기능 : 
- 홍재희 : 휴게소 상세페이지
  - 담당 기능 : 
- 차유지 : 주문결제, 로그인, 회원가입
  - 담당 기능 :
